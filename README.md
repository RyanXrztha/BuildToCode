BuildToCode

Welcome to BuildToCode, a beginner-friendly website designed to teach HTML and CSS, created by Aryan Shrestha as a solo project at Islington College, Bhaktapur. This platform aims to empower Nepali learners with interactive tutorials, local examples, and a structured learning path.

Overview

BuildToCode is an educational web development tool featuring:





Interactive slider for visual engagement.



HTML and CSS tutorials for beginners.



Contact form for user feedback.

Developed using VS Code and managed with a GitHub workflow, this project showcases a branch-based structure to handle features and testing efficiently.

Features





Slider: A dynamic image slider to greet users and highlight content.



Tutorials: Step-by-step guides for HTML and CSS basics.



Contact Form: A simple form for users to send messages.

Getting Started

Prerequisites





A web browser (e.g., Chrome).



Text editor (e.g., VS Code).



Git installed for cloning the repository.

Installation





Clone the repository:

git clone repository name.



Navigate to the project folder:

cd buildtocode



Open with VS Code or your preferred editor.



Open index.html in a browser to view the site.

Usage





Explore the slider on the homepage for an introduction.



Follow tutorials under the designated section.



Use the contact form to send feedback (currently static; backend TBD).

Project Structure





main: Stable, production-ready code.



dev: Integration of new features from sub-branches.



int: Testing combined features.



test: Experimental changes and testing.



Sub-branches: e.g., slider-structure, slider-styles for modular development.
