Mastering HTML

HTML (HyperText Markup Language) is the foundation of every website. It structures content using tags and elements, making it easy for browsers to display text, images, and links.

Essential HTML Elements

  • <header>: Defines the top section of a page, often containing a logo, navigation, or title.
  • <footer>: Represents the bottom section, typically with copyright, contact info, or links.
  • <p>: Creates a paragraph for text content, ideal for readable blocks of information.
  • <br>: Inserts a single line break within text, useful for addresses or poems.
  • <marquee>: Displays scrolling text (deprecated but functional for basic animations).
  • <div>: A container for grouping elements, styled with CSS for layout.
  • <span>: An inline container for styling specific parts of text.
  • <a>: Creates hyperlinks to other pages or sections.

Example: Using HTML Elements

This example demonstrates a styled webpage with a header, navigation, paragraph, marquee, and footer, using the BuildToCode color scheme:

<!DOCTYPE html> <html> <head> <title>My Website</title> <style> header { background: #2a0944; color: white; padding: 1rem; text-align: center; } nav a { color: #9b59b6; margin: 0 0.5rem; text-decoration: none; } nav a:hover { text-decoration: underline; } .highlight { background: #e67e22; color: white; padding: 2px 6px; border-radius: 3px; } marquee { background: #9b59b6; color: white; padding: 0.5rem; } footer { background: #2a0944; color: white; padding: 0.8rem; text-align: center; } </style> </head> <body> <header> <h1>Welcome to My Site</h1> <nav> <a href="#home">Home</a> | <a href="#about">About</a> </nav> </header> <div> <p>This is a paragraph. <span class="highlight">Highlight this text!</span></p> <p>Line one.<br>Line two after a break.</p> <marquee>Scroll this message!</marquee> </div> <footer> <p>© 2025 My Website. Contact: <a href="mailto:info@example.com">info@example.com</a></p> </footer> </body> </html>

Advanced HTML Tips

  • Use <div> for layout sections (e.g., containers for sliders or forms).
  • Apply <span> for small styling changes, like highlighting a word.
  • Ensure <a> tags have meaningful href attributes (e.g., URLs or section IDs).
  • Replace <marquee> with CSS animations for modern websites.
  • Use semantic HTML (e.g., <header>, <footer>) for accessibility and SEO.

Mastering HTML Tables

HTML tables are used to display data in a structured grid of rows and columns, ideal for presenting information like company details, schedules, or financial data. Below, we explore how to create and style tables, with examples featuring Nepali companies.

How to Use HTML Tables

Tables are created using the <table> element. Rows are defined with <tr> (table row), and cells use <td> (table data) for content or <th> for headers. Headers are typically bold and centered, used for column or row titles. Additional elements like <thead>, <tbody>, <tfoot>, and <caption> enhance structure and accessibility.

Mastering CSS

CSS (Cascading Style Sheets) brings your HTML to life by controlling colors, layouts, fonts, and animations. It can be applied inline, internally, or externally.

Core CSS Concepts

  • Selectors: Target elements (e.g., p, .class, #id).
  • Properties: Define styles like color, padding, or display.
  • Box Model: Includes content, padding, borders, and margins for each element.
  • Flexbox/Grid: Modern layout systems for responsive designs.

Example: Styling HTML Elements

The slider uses a purple and orange theme with flexbox for layout and smooth transitions for slide changes.

header { background: linear-gradient(to right, #2a0944, #3e1f5b); color: white; padding: 1.5rem; text-align: center; } p { font-size: 1.2rem; line-height: 1.8; margin: 1rem 0; } span { background: #e67e22; padding: 2px 6px; border-radius: 3px; } a { color: #9b59b6; text-decoration: none; } a:hover { text-decoration: underline; } marquee { background: #9b59b6; color: white; padding: 0.8rem; } footer { background: #2a0944; color: white; padding: 1.5rem; }

How to Code: Save this as styles.css, link it to your HTML with <link rel="stylesheet" href="styles.css">, and test with the HTML example above. This styles headers, paragraphs, spans, links, marquees, and footers.

This styles the slider which has 3 slides.

.slider { position: relative; overflow: hidden; border-radius: 12px; } .slides { display: flex; transition: transform 0.5s ease-in-out; } .slide { min-width: 100%; height: 450px; background-size: cover; position: relative; } .slide-caption { background: rgba(155, 89, 182, 0.8); color: white; padding: 1rem; text-align: center; } .slider-btn { background: #e67e22; color: white; border: none; padding: 1rem; border-radius: 50%; } .slider-btn:hover { background: #d35400; }

How to Code: This CSS creates a responsive slider with a purple caption background and orange buttons. JavaScript (below) handles slide navigation.

Advanced CSS Tips

  • Use flex or grid for complex layouts like sliders or navigation bars.
  • Apply transition for smooth animations (e.g., slider movement or hover effects).
  • Use gradients (e.g., linear-gradient) for modern backgrounds.
  • Ensure responsive design with media queries for mobile compatibility.

Tips for Using Border, Margin, and Padding

  • Use border to define element boundaries; combine with border-radius for rounded edges.
  • Use margin: auto; to center block elements horizontally.
  • Apply padding to ensure content doesn’t touch borders, improving readability.
  • Avoid excessive margins/padding to prevent cluttered layouts; test on mobile with media queries.
  • Use the box model (content + padding + border + margin) to calculate total element size.
